# Skill: 数据定义产出

改数据模型必须同步更新 `docs/<backend>/<svc>/data-model.md`。

## 每处变更需记录
1. 表用途 + 字段（名/类型/可空/默认/含义）
2. 索引（列、唯一性、用途）
3. 约束与外键
4. 迁移说明（是否可回滚、数据兼容性、是否需回填）

## 反模式
- ❌ 加字段不写文档与迁移说明
- ❌ 迁移脚本不可回滚却未标注
