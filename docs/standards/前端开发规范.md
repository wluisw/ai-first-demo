# 前端开发规范

## 技术栈
- Node 20、TypeScript 5(`strict: true`)、React 18、Vite 5、pnpm workspace。
- 测试:Vitest + Testing Library;lint:ESLint(`--max-warnings 0`);格式化:Prettier。

## 硬约束
- 禁 `any`;组件用函数组件 + hooks,禁 class component。
- 每个新功能藏在特性开关后(`flags/feature-flags.ts` + `apps/web/src/lib/flags.ts`)。
- UI 边界用 Error Boundary 兜底;网络请求统一封装,失败给可读提示,不吞错。
- 金额 / 精度敏感数值:API 返回 `string`,前端禁浮点运算(见 `financial-numerics` skill)。
- 遥测走结构化事件(`apps/web/src/lib/telemetry.ts`),字段对齐后端 slog(service / level / 事件名)。

## 命令
`pnpm dev` / `pnpm build` / `pnpm test` / `pnpm typecheck` / `pnpm lint`(或根 `make web-*`)。
